# Игра «Арифметическая прогрессия» (Laravel)

## Описание
SPA-игра, в которой пользователь видит последовательность чисел с пропуском и должен вставить правильное значение. Проект реализован на Laravel (бэкенд API) и чистом JavaScript (фронтенд).

## Установка и запуск

1. Перейдите в директорию проекта:
   ```bash
   cd Task04
   ```
2. Выполните установку одной командой:
    ```bash
   make install
   ```
Эта команда:

- установит зависимости Composer

- создаст файл .env из примера

- сгенерирует ключ приложения

- создаст файл базы данных database/database.sqlite

- выполнит миграции (создаст таблицы)
3. Запустите встроенный веб-сервер Laravel:
    ```bash
   php artisan serve
   ```
   или используйте make serve.
4. Откройте в браузере: http://localhost:8000

Требования
- PHP 8.2

- Composer

- SQLite (включён в PHP)
